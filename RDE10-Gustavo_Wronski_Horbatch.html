Cascading Style Sheets

Cascading Style Sheets (CSS)- "Folhas de Estilo em Cascata",é o responsável pela estilização
de um documento HTML.

Define dentre outras coisas:

  • Fontes
  • Cores
  • Espaçamentos
  • Tamanhos

Ou seja, o HTML realiza a criação do conteúdo, já o CSS é o responsável pela estilização do mesmo:

Formas de utilizar o CSS

Inline

Exemplo:

  1. Os códigos CSS são definidos dentro de uma Tag HTML.
  2. Afetam específicamente a tag em que foi definida.

Incorporado

  1. Os códigos CSS são definidos no cabeçalho, na teg head.
  2. Afeta apenas o documento em que foi definido.

Externo

  1. Os códigos CSS são definidos em um documento externo.
  2. Afeta todos os documentos que estão linkados no CSS.

Envelopamento

Dessa forma é possível selecionar uma parte do documento que ainda não foi definida
com tag (uma parte avulsa) e insira-a em uma tag.

Como realizar o envelopamento:

  1. Selecione a parte.
  2. Utilize as teclas CTRL+SHIFT+P.
  3. Seleciona a opcção: Quebrar linha com abreviação.
  4. Digite a tag que vai ser envolvida na parte selecionada.
  5. Clique em ENTER.
