HIPERLINKS

Hiperlink ou link

É uma referência dentro de um documento de hipertexto para outras partes do mesmo documento,
outros documentos, arquivos ou endereço eletrônico.

OS LINKS PODEM SER:

  • Remotos
  • Internos
  • Locais
  • Com endereço de correio
  • Arquivos

MARCAÇÃO DE HIPERLINKS

Utiliza-se a TAG a e a propriedade href, que faz referência ao conteúdo desejado.

SINTAXE:

HIPERLINKS EXTERNOS ou REMOTOS

São links que referenciam SITES ou PÁGINAS DA WEB. Na propriedade href utiliza-se a URL
(Uniform Resource Locator) para a referência desejada.

SINTAXE:

HIPERLINKS INTERNOS

São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar
diferentes parágrafos ou seções quando o documento é extenso.

SINTAXE:


HIPERLINKS LOCAIS

São links que apontam para outras páginas dentro de um mesmo site.

SINTAXE:


HIPERLINKS EM IMAGENS

É possível realizar hiperlinks imagens envolvendo a tag de imagem com a tag de hiperlink

SINTAXE:

HIPERLINKS PARA DOWNLOADS DE ARQUIVOS

São links que apontam para arquivos e possibilitam realizar download destes arquivos.

SINTAXE:


HIPERLINKS PARA ENDEREÇO ELETRÔNICO

São links que apontam para um endereço eletrônico
