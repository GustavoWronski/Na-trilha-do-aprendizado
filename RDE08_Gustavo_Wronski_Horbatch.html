TAGS

<table>

Essa TAG determina uma tabela HTML, essa TAG utiliza elementos para forma-la.

  • <table>tabela<table> Define a tabela
  • <tr>linha</tr> Define linha
  • <th>coluna</th> Define coluna
  • <td>dados</td> Define célula

Dessa forma, a tabela será exibida sem borda.Para incluir borda, é necessário incluir o comando border da seguite forma:

border: 1px solid black

O número de px influencia na grossura da borda.

<del>

Tag que define o texto excluido do documento, com um risco sobreposto.

Sintaxe:

<del>texto a ser riscado</del>

<ins>

Tag que define a inserção de um novo texto no documento, ele estará sinalizado de forma sublinhada.

Sintaxe:

<ins>texto a ser inserido</ins>

<sub>

Tag que define um texto abaixo da linha normal (subscrito)

Sintaxe:

<sub>texto a ser subscrito</sub>

<sup>

Tag que define um texto acima da linha normal (sobrescrito)

Sintaxe:

<sup>texto a ser sobrescrito</sup>

<blockquote>

Tag que define um trecho citado de uma fonte externa extensa

Sintaxe:

<blockquote>citação externa longa</blockquote>

<q>

Tag que define um trecho citado de uma fonte pequena

Sintaxe:

<q>citação externa curta</q>

<abbr>

Tag que define a abreviação de palavra e, quando está junta da tag <title> exibirá a palavra completa

Sintaxe:

<abbr title="texto completo">texto abreviado<abbr>

<address>

Tag que define as opções cde contato com o proprietário do site ou documento

Sintaxe:

<address>informação de contato</address>

<cite>

Tag que define título

Sintaxe:

<cite>título</cite>

<bdo>

Tag que define a direção do texto

Sintaxe:

<bdo>texto</bdo>

<code>

Tag que define que o texto é o pedaço de um código, desse modo o conteúdo é exibido em fonte monoespaçada

Sintaxe:

<code>pedaço de código</code>

<pre>

Tag que define o uso do texto igual ao do código fonte, mantendo absolutamente tudo.

Sintaxe:

<pre>texto identico ao código fonte</pre>

<kbd>

Tag que define a entrada de texto por parte do usuário. O conteúdo é exibido em fonte monoespaçada.

Sintaxe:

<kbd>entrada de texto pelo usuário</kbd>
