Pelo URL da página, é mandada uma requisição para um servidor DNS, que por sua vez busca o IP do site, que é retornado para o cliente, consequentemente o cliente dispara uma requisição ao servidor do site, que depois disponibiliza os arquivos HTML, CSS e JAVA SCRIPT para o cliente, cada um dos arquivos desempenha um papel, juntos formando o site.

PÁGINA WEB e WEBSITE

PÁGINA WEB é um arquivo que possibilita a inserção de: TEXTOS, IMAGENS, TABELAS, PLANILHAS, APRESENTAÇÕES, VÍDEOS que tenham ligações de hipertexto que permitem a transição de uma página para outra

WEBSITE é uma coleção de páginas web, armazenadas em diversas pastas vinculadas a um servidor

O sistema funciona em LADO DO CLIENTE e LADO DO SERVIDOR, onde o cliente realiza uma requisição e o servidor processa o mesmo e retorna ao cliente. Para isso são necessários profissionais.

HTML

Linguagem de Marcação de Hipertexto

Baseada em TAGS, criada na década de 90, por TIM BERNERS-LEE, possibilitou a publicação de conteúdo em hipertexto, podendo ser vinculadas por hyperlinks

TAGS

Definem a MARCAÇÃO DO CONTEÚDO, cada tag tem funcionalidades distintas. Podem possuir ATRIBUTOS, como parâmetros para o navegador.Sem o uso das mesmas é exibido um texto plano.

O início do programa se dá com a tag HTML, dentro desta HEAD, que define o cabeçalho, dentro da mesma TITLE, que define uma barra de título para o navegador e dentro desta BODY, onde fica o conteúdo.

Para adicionar comentários, usa-se o parênteses angulado, sinal de exclamação e dois traços, depois do comentário dois traços e fecha-se o parênteses angulado

DOCTYPE

É a TAG que declara o tipo do documento.

Dentro da tag HEAD, indroduz-se as tags META, que definem por exemplo, o tipo de tela utilizada

MARCAÇÃO DE TÍTULOS

Utiliza-se as tags HEADING (h), de 1 até 6

MARCAÇÃO DE PARÁGRAFOS

Utiliza-se as tags de PARÁGRAFO (p), para iniciar e finalizar parágrafos

MARCAÇÃO DE QUEBRA DE LINHA

Utiliza-se as tags de QUEBRA DE LINHA (br)
