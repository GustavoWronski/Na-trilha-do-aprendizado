HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee.

A HTML é utilizada para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo , e que podem ser vinculados através de hyperlinks.

  • 1990- Criação do HTML
  • 1995- Versão 2.0
  • 1997- Versão 3.2
  • 1999- Versão 4.1
  • 2014- HTML 5

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

MARCAÇÃO DE LISTAS

Listas não-ordenadas

Usa-se a tag de marcação com as letras ul entre parenteses angulares.
Depois cada item fica entre as letras li, também entre parenteses angulares.

Exemplo:
  • Item 1
  • Item 2
  • Item 3

Lista ordenada

Utiliza-se a ted de marcação com as letras ol entre parenteses angulares
Depois cada item fica entre as letras li, também entre parenteses angulares.
Neste tipo de lista ainda é possível indicar o elemento que será usado para ordenar os itens(letras ou números), usando "type" e o elemento entre parenteses logo após o "ol".

Exemplo:
  1. Primeiro
  2. Segundo
  3. Terceiro

Lista detalhada

Utiliza-se a tag de marcação com as letras dl entre parenteses algulares
E em seguida cada item entre as letras dl, também entre parenteses angulares.
Para realizar o detalhamento de cada item repete-se o mesmo processo indicado acima, mas desta vez com as letras dd.

Exemplo:
Primeiro lugar
100 pontos
Segundo lugar
75 pontos
Terceiro lugar
45 pontos
